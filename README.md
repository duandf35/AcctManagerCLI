**Requirements:**

Install SQLite engine: https://www.sqlite.org/download.html

Show help information
```
am -h
```

Add new account
```
am -a label username password
```
Update username
```
am -u label new_username
```
Update password
```
am -p label new_password
```
List
```
am -l
```
Find
```
am -f label
```

Dependencies:

sqlite3
